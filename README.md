# Moon Music Website

A lunar themed public landing page for Moon Music Bot.

## Deploy on Vercel

Import this repository in Vercel. Framework preset: Next.js.

Optional environment variables:

```env
NEXT_PUBLIC_INVITE_URL=https://discord.com/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=37080128&scope=bot%20applications.commands
NEXT_PUBLIC_SUPPORT_SERVER=https://discord.gg/your-server
```

Do not add your bot token or private config to this website.
